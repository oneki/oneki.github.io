Installation

Oneki.js is a React framework based of the following components:

  • React
  • React Router
  • Redux
  • Redux Saga
  • Immer

The goal of Oneki.js is to propose a framework on top of these librairies that hides most of the complexity. Therefore, the developer can built more quickly powerful webapps respecting the best practices.

#Installation

Create a new React application with Create React App and add onekijs as a dependency:

yarn create react-app my-app
cd my-app
yarn add onekijs
yarn install

#Next steps

Create your first basic application

To fully master the framework, be sure to read the documentation of

  • React
  • React Router
  • Redux
  • Redux Saga

Oneki.js has an opinionated approach on how to use these librairies and tries to remove most of the boilerplate, but it's important to understand how it works underneath.
